The **MotorVista** server-side application powers the car rental platform by handling all backend operations, including vehicle management, user authentication, booking functionality, and secure payments. It provides a robust API that communicates with the frontend to allow users to browse, book, and rent vehicles seamlessly.

- **Node.js**: JavaScript runtime environment used for building the backend API.
- **Express.js**: A lightweight web application framework for Node.js.
- **MongoDB**: A NoSQL database used for storing user data, vehicle details, and booking information.
- **react-router-dom**: Handling API requests and routing.
- **react-toastify**: For displaying toast notifications like booking confirmations or errors.

## üîë **Key Features**
- **User Authentication**: Secure user sign-up, login, and session management.
- **Vehicle Management**: CRUD operations for vehicle listings (add, edit, delete, view).
- **Booking System**: Users can view real-time vehicle availability and make bookings.
- **Payment Integration**: Secure payment processing for car rentals.
- **Email Notifications**: Automatic booking confirmation and payment receipt emails.
